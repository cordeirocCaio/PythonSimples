# Introdução ao Python para Agentes IA

Este projeto é uma introdução prática ao desenvolvimento de agentes de inteligência artificial usando Python. Aqui, você aprenderá os conceitos básicos de programação em Python aplicados à criação de agentes IA do zero.

## Pré-requisitos

- Python 3.8 ou superior instalado.
- Conhecimento básico de programação (opcional, mas recomendado).

## Instalação

1. Clone este repositório:
   ```
   git clone https://github.com/seu-usuario/crie-rapido-agentes-ia.git
   ```

2. Navegue até a pasta do projeto:
   ```
   cd crie-rapido-agentes-ia/python/introducao
   ```

3. Instale as dependências (se houver):
   ```
   pip install -r requirements.txt
   ```

## Uso

Execute os scripts de exemplo para começar:

```
python exemplo_basico.py
```

## Estrutura do Projeto

- `exemplo_basico.py`: Script introdutório com conceitos fundamentais.
- `agente_simples.py`: Implementação de um agente IA simples.
